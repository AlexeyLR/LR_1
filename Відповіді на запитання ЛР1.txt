1.Що таке Git? 
   GIT - розподілена система керування версіями файлів та спільної роботи.
2.В якому статусі можуть перебувати файли при роботі з Git?
   Щоб дізнатись, в якому стані файли, варто скористатись командою git status. Ця команда показує вам, в якій ви зараз гілці та інформує вас про те, що вона не розбіглася з такою ж гілкою на сервері. Поки що, ця гілка завжди буде “master”, така гілка створюється автоматично.
3.Для чого використовується  команда add? 
   Контролювання нових файлів(добавлення файлів)
4.Для чого використовується  команда status?
   Перевірка статусу файлів.(Можна дізнатися стан файлів)
5.Для чого використовується  команда commit? 
   Створення коміту, тобто ця програма може запустити редактор. 
6.Для чого використовується  команда init? 
   Створення репозиторія.
7.Що необхідно писати в коментарях комітів? 
  Якщо відбулися якісь зміни.
8.Як переглянути список комітів? 
   git log перераховує коммітов, зроблені в репозиторії в зворотному до хронологічного порядку - останні коммітов знаходяться вгорі.
9.Для чого використовується  команда checkout?
   Для переходу на іншу гілку.
